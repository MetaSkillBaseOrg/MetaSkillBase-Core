# MetaSkillBase-Core Glossary

**Version:** 1.0.2 (2026-03-15)

- **Meta Skill**: Atomic, reusable, finest-grained skill unit.
- **skill-decomposer**: Demand decomposition and skill search tool (v1.0.2).
- **Contributor**: Anyone who submits skills, demands, code, docs.
- **Usage Count**: Community metric reflecting skill value.
- **Elastic Classification**: Organize only, no rejection, no audit; existing directories are examples only.
- **OpenClaw Compatible**: Works with OpenClaw.
- **One for Human, One for Bot**: Every product needs two instructions — one for humans, one for bots.
- **Supply-Driven**: Every Meta Skill is a valuable "supply" — the more, the better.
- **Demand-Responsive**: Every demand can be decomposed and fulfilled.
- **Human-Bot Symbiosis**: Humans describe needs; machines execute solutions.
- **Quick Submit**: Submit demands via proxy server (http://47.79.18.9:3000).
- **Manual Submit**: Submit demands via GitHub issue link.
- **Flexible Classification**: Encourage creation of new categories; existing directories are samples.
