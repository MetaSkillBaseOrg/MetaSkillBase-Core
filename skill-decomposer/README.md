# Skill Decomposer

Skill Decomposer is a component of the OpenClaw ecosystem, used to decompose complex skills into atomic, reusable sub-skills that comply with CloudHub and ClawHub specifications.

## Features
- Decompose complex skills into independent, reusable sub-skills
- Comply with CloudHub & ClawHub publishing standards
- Support automatic validation and formatting
- Provide stable runtime and service support
- Integrate with OpenClaw ecosystem seamlessly

## Project Structure
```
skill-decomposer/
├── SKILL.md
├── README.md
└── scripts/
    ├── device-flow-server.js
    └── device_flow.js
```

## Usage
This module is designed to run within the OpenClaw environment and can be published and used after registering and authenticating on CloudHub.

## Environment Requirements
- OpenClaw runtime
- Node.js supported environment
- CloudHub / ClawHub account
