# CONTRIBUTING

**Version:** 1.0.2 (2026-03-15)

## How to Contribute

### 1. Submit a Demand
Use skill-decomposer to analyze your requirement. If no existing skill satisfies your need, submit a demand to the community.

### 2. Develop a Skill
- Skills must be atomic: one skill = one minimal function
- Follow OpenClaw skill structure
- Include complete documentation

### 3. Classification Rules (Important)
- Directories under skills/ are **examples only**, NOT mandatory.
- You may use existing directories or create new classification directories freely.
- If unsure about classification, place your skill into 00-Unclassified.
- No audit or rejection of classification choices.

### 4. Submit Your Work
- Fork the repository
- Add your skill under skills/ (any directory, existing or new)
- Submit a Pull Request
- PR will be merged automatically after 7 days if no community objections

## Standards
- Atomic: one skill = one minimal function
- Naming: lowercase with hyphens
- OpenClaw compatible
- Each skill has complete documents
